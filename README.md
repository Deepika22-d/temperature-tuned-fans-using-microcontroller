## **🌿 PROJECT OVERVIEW**

An intelligent automatic fan control system designed to adjust fan speed based on real-time temperature readings using a microcontroller and temperature sensor.

This smart cooling solution helps to:

✔ Maintain optimal room temperature  
✔ Reduce electricity consumption  
✔ Improve user comfort  
✔ Eliminate manual speed adjustment  
✔ Enhance overall energy efficiency  

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## **🌡 PROBLEM STATEMENT**

Conventional fan systems have several limitations:

• Manual speed control is inconvenient  
• No automatic response to temperature changes  
• Continuous operation leads to energy wastage  
• Inconsistent cooling in changing environments  

A temperature-based automated control system is required for efficient and responsive cooling.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## **💡 PROPOSED SOLUTION**

The system integrates smart control components:

🌡 Temperature Sensor (DHT11 / LM35)  
🧠 Microcontroller (Arduino / ESP32)  
⚙ PWM-Based Speed Regulation  
🔌 MOSFET Motor Driver Circuit  
📟 Optional LCD / IoT Monitoring  

✔ Fan speed automatically increases when temperature rises  
✔ Fan speed decreases when temperature drops  
✔ Ensures energy-efficient and adaptive cooling

## **⚙️ SYSTEM WORKFLOW**

1️⃣ Temperature sensor continuously monitors room temperature  
2️⃣ Microcontroller reads temperature data  
3️⃣ Compare value with predefined threshold levels  
4️⃣ Generate PWM signal based on temperature  
5️⃣ Control fan speed using MOSFET driver  
6️⃣ Display temperature and fan status  
7️⃣ Optional: Send data to IoT platform
